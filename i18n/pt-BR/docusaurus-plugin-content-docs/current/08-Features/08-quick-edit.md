O recurso de Edição Rápida (Quick Edit) no Phoenix Code permite que você edite arquivos CSS diretamente dentro de arquivos HTML. Essa integração ajuda você a se concentrar em um arquivo de cada vez, reduzindo distrações. Usar a Edição Rápida com a função de Visualização ao Vivo (Live Preview) fornece feedback imediato sobre suas alterações, permitindo que você ajuste seu CSS de forma eficiente.

## Como Usar a Edição Rápida

Siga estas etapas para usar o recurso de Edição Rápida:

1. **Posicione o Cursor**: Coloque o cursor onde você precisa iniciar a Edição Rápida no arquivo HTML.

   ![Posicione o cursor para Edição Rápida](images/QuickEdit/quickedit.png)

2. **Acesse a Edição Rápida**: Você pode acessar a Edição Rápida de uma de três maneiras:
   * **Atalho de Teclado**: Pressione `Ctrl + E` (Windows/Linux) or `Cmd + E` (Mac).
   * **Atalho de Mouse**: Combine `Ctrl + clique` (Windows/Linux) ou `Cmd + clique` (Mac).
   * **Menu de Contexto**: Clique com o botão direito para abrir o menu de contexto e escolha Edição Rápida.

   ![Acesse a Edição Rápida pelo menu de contexto ou usando um atalho](images/QuickEdit/quickeditcontext.png)

3. **Edite o CSS**: A janela de Edição Rápida aparecerá, mostrando o CSS relacionado ao elemento HTML selecionado.

   ![A janela de Edição Rápida](images/QuickEdit/quickeditPanel.png)

4. **Salvar e Fechar**: Depois de fazer as alterações necessárias, salve suas edições e feche a janela de Edição Rápida para continuar seu trabalho.
