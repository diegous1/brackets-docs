# Guia de Configuração do Phoenix Code

## Introdução

O Phoenix Code é um editor de texto projetado para tornar a programação tão intuitiva e divertida quanto jogar um videogame – criado especialmente para desenvolvedores web, designers e estudantes. Este guia fornece instruções passo a passo sobre como baixar, instalar e começar a usar o Phoenix Code no seu sistema.

## Passo 1: Baixando o Phoenix Code

Visite o site oficial – [phcode.io](https://phcode.io) e clique no botão de download para baixar o instalador apropriado para o seu sistema operacional.

![Alt text](images/Website/download.png)

NB: Usuários Linux, consulte a seção abaixo `Como Instalar no Linux`

### Escolhendo um Instalador Personalizado

Se você precisar de uma versão para um sistema operacional diferente ou tiver requisitos específicos:

1. Clique no menu suspenso **"Download"** no site.
2. Selecione o instalador desejado entre as opções disponíveis.

![Alt text](images/Website/downloadOptions.png)

O Phoenix Code é compatível com uma ampla gama de sistemas operacionais, incluindo:

- **Mac**: chipsets M1 e Intel
- **Windows**: arquiteturas x64
- **Linux**: Ubuntu, Pop!_OS, Fedora, Arch, e mais

## Passo 2: Instalando o Phoenix Code

Execute o instalador baixado (no Windows e Mac) e siga as instruções na tela para instalar o Phoenix Code no seu dispositivo.

### Como Instalar no Linux

![Alt text](images/Website/linuxCmd.png)
1. Abra o Terminal. Você geralmente pode encontrá-lo no menu de aplicativos.
1. Copie o comando do site ou use o comando:
```bash
wget -qO- https://updates.phcode.io/linux/installer.sh | bash
```


## Usando a Versão Web

Se você preferir não instalar nenhum software, pode usar a versão web do Phoenix Code acessando [https://phcode.dev](https://phcode.dev). Esta versão permite usar o Phoenix Code diretamente no seu navegador.

## Começando

Com o Phoenix Code instalado ou acessado pela web, você está pronto para mergulhar em seus projetos de programação e design web.
