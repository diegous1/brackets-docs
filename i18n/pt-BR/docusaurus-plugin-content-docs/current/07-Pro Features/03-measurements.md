O recurso de **Medições** exibe linhas de régua que se estendem das bordas de um elemento selecionado até as bordas do documento do Live Preview, com rótulos que mostram as posições exatas em pixels. Isso ajuda você a posicionar os elementos com mais precisão.

![Medições](../images/measurements.png "Medições")

> As linhas da régua aparecem em azul para elementos editáveis e em cinza para elementos não editáveis (criados dinamicamente).

---

## Ativando as Medições

Para ativar as medições, clique no **menu suspenso do seletor de modo** na barra de ferramentas do Live Preview e selecione **Show Measurements**. Por padrão, esta opção está desmarcada.

![Ativar Exibir Medições](../images/enable-show-measurements.png "Ativar Exibir Medições")

Como alternativa, você pode ativar as medições no menu **Mais Opções** (ícone de três pontos) na Caixa de Ferramentas. Selecione **Show Measurements** para ativar ou desativar.

Você também pode alterar essa configuração atualizando a preferência `livePreviewShowMeasurements` no arquivo de preferências. Defina-a como `true` or `false` (padrão). Consulte [Editando Preferências](../editing-text#editing-preferences) para aprender como editar o arquivo de preferências.
